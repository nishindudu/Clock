# A Simple Analogue Clock

A simple analogue clock built with HTML, CSS, and JavaScript. It features customizable update frequency and background color settings.

## Features
- Real-time analogue clock display
- Customizable update frequency
- Background color selection

## Demo
A live demo of the clock can be found [here](https://nishindudu.github.io/Clock/).

## Technologies Used
- HTML5
- CSS3
- JavaScript
